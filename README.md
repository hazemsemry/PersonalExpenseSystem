\# Personal Expense System



Progetto per la gestione delle spese personali.



\## Requisiti

\- Python 3



\## Come eseguire



Aprire il terminale nella cartella `src` e scrivere:



python main.py



\## Funzionalità

\- Gestione categorie

\- Inserimento spese

\- Definizione budget

\- Report:

&#x20; - Totale per categoria

&#x20; - Spese vs budget

&#x20; - Lista completa delle spese



\## Database

Il database è gestito tramite SQLite con le seguenti tabelle:

\- categories

\- expenses

\- budgets
